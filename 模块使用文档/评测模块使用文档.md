# 评测模块使用文档

## 1. 模块结构

评测模块位于 `OJ/judge` 目录下，采用模块化设计，包含以下核心组件：

```
judge/
├── include/          # 头文件目录
├── src/              # 源文件目录
│   ├── compiler.cpp         # 编译器实现
│   ├── judge_engine.cpp     # 评测引擎实现
│   ├── language_config.cpp  # 语言配置实现
│   ├── main.cpp             # 主程序入口
│   ├── result_judger.cpp    # 结果判定器实现
│   ├── sandbox.cpp          # 沙箱环境实现
│   └── submission_handler.cpp # 提交处理实现
├── config/           # 配置文件目录
│   └── languages.json       # 语言配置文件
├── tests/            # 测试文件目录
├── build/            # 构建目录
└── CMakeLists.txt    # CMake配置文件
```

## 2. 核心组件

### 2.1 编译器（Compiler）

负责将源代码编译为可执行文件，支持C、C++、Java和Python四种语言。

**主要类**：
- `Compiler`：编译器基类，定义了编译接口
- `CCompiler`：C语言编译器实现
- `CppCompiler`：C++语言编译器实现
- `JavaCompiler`：Java语言编译器实现
- `PythonCompiler`：Python语言编译器实现（解释型语言，无需编译）

**使用方法**：
```cpp
#include "compiler.h"

// 创建编译器实例
CCompiler c_compiler;

// 编译代码
std::string code = "#include <stdio.h>\nint main() { printf(\"Hello\"); return 0; }";
CompileResult result = c_compiler.compile(code, "test.exe");

// 处理编译结果
if (result.success) {
    // 编译成功
} else {
    // 编译失败，查看错误信息
    std::cout << result.output << std::endl;
}
```

### 2.2 沙箱（Sandbox）

提供安全的代码执行环境，限制资源使用和访问权限。

**主要类**：
- `Sandbox`：沙箱类，用于安全执行用户代码
- `SandboxConfig`：沙箱配置结构体，设置时间限制、内存限制等参数

**使用方法**：
```cpp
#include "sandbox.h"

// 创建沙箱配置
SandboxConfig config;
config.time_limit_ms = 1000;  // 1秒时间限制
config.memory_limit_mb = 128; // 128MB内存限制

// 创建沙箱实例
Sandbox sandbox(config);

// 执行命令
SandboxResult result = sandbox.execute("echo hello", "");

// 处理执行结果
std::cout << "Output: " << result.output << std::endl;
std::cout << "Exit code: " << result.exit_code << std::endl;
std::cout << "Runtime: " << result.runtime_ms << "ms" << std::endl;
```

### 2.3 结果判定器（ResultJudger）

用于比较程序输出和预期输出，判定评测结果。

**主要类**：
- `ResultJudger`：结果判定器类

**使用方法**：
```cpp
#include "result_judger.h"

// 创建判定器实例
ResultJudger judger;

// 判定结果
std::string expected_output = "Hello\n";
std::string actual_output = "Hello\n";
JudgeResult result = judger.judge(actual_output, expected_output);

// 处理判定结果
std::cout << "Result: " << result.status << std::endl;
if (result.status != "AC") {
    std::cout << "Message: " << result.message << std::endl;
}
```

### 2.4 评测引擎（JudgeEngine）

整合编译、执行、判定流程，提供完整的评测功能。

**主要类**：
- `JudgeEngine`：评测引擎类

**使用方法**：
```cpp
#include "judge_engine.h"

// 创建评测引擎实例
JudgeEngine engine;

// 准备评测数据
std::string code = "#include <stdio.h>\nint main() { printf(\"Hello\"); return 0; }";
std::string language = "c";
std::string input = "";
std::string expected_output = "Hello";

// 执行评测
JudgeResult result = engine.judge(code, language, input, expected_output);

// 处理评测结果
std::cout << "Status: " << result.status << std::endl;
std::cout << "Score: " << result.score << std::endl;
std::cout << "Runtime: " << result.runtime_ms << "ms" << std::endl;
std::cout << "Memory: " << result.memory_kb << "KB" << std::endl;
```

### 2.5 语言配置（LanguageConfig）

管理支持的编程语言配置，从配置文件加载语言信息。

**主要类**：
- `LanguageConfig`：语言配置类

**使用方法**：
```cpp
#include "language_config.h"

// 加载语言配置
LanguageConfig config;
config.load("config/languages.json");

// 获取语言信息
LanguageInfo c_info = config.getLanguage("c");
std::cout << "C compiler: " << c_info.compiler << std::endl;
std::cout << "Compile command: " << c_info.compile_command << std::endl;
```

## 3. 配置说明

### 3.1 语言配置文件

语言配置文件位于 `config/languages.json`，包含支持的编程语言信息：

```json

      "source_extension": ".c",
      "output_extension": ".out",

    "cpp": {

      "name": "Java",
      "version": "Java 17",
      "compiler": "javac",
      "compile_command": "javac {source}",
      "source_extension": ".java",
      "output_extension": ".class",
      "execute_command": "java {filename}",
      "time_limit_multiplier": 1.5,
      "memory_limit_multiplier": 1.5
    },
    "python": {
      "id": "python",
      "name": "Python",
      "version": "Python 3.10",
      "compiler": "python3",
      "compile_command": "",
      "source_extension": ".py",
      "output_extension": "",

      "memory_limit_multiplier": 1.5
    }
  }

```

### 3.2 配置参数说明

- `id`：语言标识符
- `name`：语言名称
- `version`：语言版本
- `compiler`：编译器名称
- `compile_command`：编译命令模板
- `source_extension`：源文件扩展名
- `output_extension`：输出文件扩展名
- `execute_command`：执行命令模板
- `time_limit_multiplier`：时间限制乘数
- `memory_limit_multiplier`：内存限制乘数

## 5. 集成方法

### 5.1 作为库集成

将评测模块作为静态库集成到其他项目中：

```cmake
# 在CMakeLists.txt中添加
add_subdirectory(path/to/judge)
target_link_libraries(your_project judge_core)
```

### 5.2 通过HTTP接口使用

评测模块提供HTTP接口，可通过网络请求提交代码和获取评测结果。

**提交代码接口**：
- URL: `/submit`
- Method: POST
- Body: JSON格式，包含code、language、input、expected_output等字段

**查询结果接口**：
- URL: `/result/{submission_id}`
- Method: GET

## 6. 注意事项

1. **环境依赖**：
   - C/C++：需要安装gcc/g++
   - Java：需要安装JDK
   - Python：需要安装Python解释器

2. **安全考虑**：
   - 沙箱环境用于限制用户代码的资源使用和访问权限
   - 建议在生产环境中进一步加强安全措施

3. **性能优化**：
   - 对于大量评测任务，建议使用异步处理
   - 可以考虑使用线程池提高并发处理能力

4. **错误处理**：
   - 编译失败时，错误信息会包含在编译结果中
   - 运行时错误会被捕获并返回相应的错误信息

## 7. 故障排除

### 7.1 编译失败

- 检查编译器是否正确安装
- 检查源代码是否有语法错误
- 检查编译命令是否正确

### 7.2 运行超时

- 检查代码是否存在无限循环
- 调整时间限制参数

### 7.3 内存超限

- 检查代码是否存在内存泄漏
- 优化算法，减少内存使用
- 调整内存限制参数

### 7.4 结果错误

- 检查代码逻辑是否正确
- 检查输入输出格式是否符合要求
- 检查预期输出是否正确

## 8. 示例代码

### 8.1 完整评测示例

```cpp
#include "judge_engine.h"
#include <iostream>

int main() {
    // 创建评测引擎
    JudgeEngine engine;
    
    // 准备评测数据
    std::string code = R"(
#include <iostream>
using namespace std;
int main() {
    int a, b;
    cin >> a >> b;
    cout << a + b << endl;
    return 0;
}
    )";
    
    std::string language = "cpp";
    std::string input = "1 2";
    std::string expected_output = "3\n";
    
    // 执行评测
    JudgeResult result = engine.judge(code, language, input, expected_output);
    
    // 输出评测结果
    std::cout << "=== 评测结果 ===" << std::endl;
    std::cout << "状态: " << result.status << std::endl;
    std::cout << "得分: " << result.score << std::endl;
    std::cout << "运行时间: " << result.runtime_ms << "ms" << std::endl;
    std::cout << "内存使用: " << result.memory_kb << "KB" << std::endl;
    
    if (result.status != "AC") {
        std::cout << "错误信息: " << result.message << std::endl;
    }
    
    return 0;
}
```

### 8.2 自定义沙箱配置

```cpp
#include "sandbox.h"
#include <iostream>

int main() {
    // 创建自定义沙箱配置
    SandboxConfig config;
    config.time_limit_ms = 2000;      // 2秒时间限制
    config.memory_limit_mb = 256;     // 256MB内存限制
    config.output_limit_mb = 20;       // 20MB输出限制
    config.enable_network = false;     // 禁用网络
    config.enable_file_system = false; // 禁用文件系统
    
    // 创建沙箱
    Sandbox sandbox(config);
    
    // 执行命令
    std::string command = "echo \"Hello from sandbox\"";
    SandboxResult result = sandbox.execute(command, "");
    
    // 输出结果
    std::cout << "命令输出: " << result.output << std::endl;
    std::cout << "退出码: " << result.exit_code << std::endl;
    std::cout << "运行时间: " << result.runtime_ms << "ms" << std::endl;
    
    return 0;
}
```
